# Релизы

## 1 декабря 2025 г.

* Обновлена сетка тарифов на API-сервисы. Подробнее об изменениях см. в [личном кабинете](https://platform.2gis.ru/ru/tariffs?utm_source=docs&utm_medium=referral&utm_campaign=api_places).

## 31 января 2025 г.

* Поддержка метода `/2.0/catalog/branch/list` будет прекращена в соответствии с политикой обратной совместимости. Используйте метод `/3.0/items`. Подробнее см. в [Оферте](https://law.2gis.ru/offer-license-agreement-webapi).

## 20 ноября 2020 г.

* Поддержка методов `/2.0/catalog/branch/search` и `/2.0/catalog/branch/get` будет прекращена в соответствии с политикой обратной совместимости. Для получения объекта по идентификатору используйте метод `/3.0/items/byid`, а для поиска мест — метод `/3.0/items`. Подробнее см. в [Оферте](https://law.2gis.ru/offer-license-agreement-webapi).
